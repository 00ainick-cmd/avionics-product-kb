# VHF Radio Upgrades: Trig, Garmin Top Picks

**Source:** Aviation Consumer
**Author:** Larry Anglisano

**Category:** Comm Radios / VHF
**Products covered:** Garmin GTR 200/225, Becker AR-6201/6203, Trig TY96A/TY97A/TY91/TY92, Icom A220, VAL Com 2000, TKM MX11, PS Engineering PAR200B

---

This article examines standalone VHF transceiver options for aircraft upgrading from legacy comm systems. Modern buyers face decisions involving WAAS GPS integration, backup navigation needs, and space constraints.

## Installation Considerations

Before upgrading, evaluate existing antenna systems. Adding a second comm radio requires a second antenna, potentially costing $500+. Belly-mounted comm antennas are easier to install than top-mounted alternatives. Worn fiberglass antennas with degraded antistatic coating should be replaced. An audio panel is necessary when installing dual radios.

## Garmin GTR Series ($1,199-$4,095)

TSO-certified and PMA-certified versions available. GTR 200/225 models transmit 10 watts (16-watt option), weigh under two pounds, measure 1.35 inches tall. Features include 3D audio processing, Bluetooth connectivity (GTR 200B), GPS integration for automatic frequency display, and standby frequency monitoring.

## Becker Avionics ($1,300-$1,408)

Space-saving designs for tight panels. AR-6203 rack-mounted uses 2.4 inches vertical space; AR-6201 fits 2.25-inch instrument cutouts. Recently began domestic assembly, improving pricing and support.

## Trig Avionics ($1,110-$3,048)

Both stack and remote-mounted options. TY96A (10-watt, $1,865) and TY97A (16-watt, $3,048) stack models measure 1.3 inches tall. Remote versions include TY91 ($1,110) and TY92 ($2,943). Features include 30-second audio replay, Dual Watch mode, and USB memory for custom frequency databases.

## Other Options

Icom A220: 8-watt with OLED display and 180-degree viewing angle. VAL Com 2000 ($1,285): 8-watt, plug-compatible with Garmin SL-40. TKM MX11 ($1,828): replacement for older Narco radios. PS Engineering PAR200B ($3,495): combines audio panel, intercom, and remote Trig TY91L transceiver.

## Recommendations

For rack-mounted: Garmin GTR 200B and Trig TY96. For space-constrained installations: Trig TY91/92 and Becker 6201. For VHF navigation needs, Garmin's GNC 255 remains the primary option, with the GNC 355 GPS/comm as an alternative.
