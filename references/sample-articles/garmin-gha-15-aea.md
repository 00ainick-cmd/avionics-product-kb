# Garmin GHA 15 Radar Height Advisory System
**Source:** AEA Avionics News (March 2026)

The GHA 15 is a radar-based height advisory system (radar altimeter) providing precise Height Above Ground Level (AGL) information.

- **Range:** Measures 0 to 500 feet AGL.
- **Precision:** Accurate to a single foot during landing flare.
- **Aural Callouts:** Provides altitude callouts between 0 to 300 feet (e.g., "ten, nine, eight...").
- **Decision Height (DH):** Pilot-configurable warnings with "Minimums, Minimums" aural alerts.
- **Seaplane Utility:** Extremely useful for glassy water landings where judging altitude is difficult.

- **Certification:** FAA STC with 500+ aircraft AML (Part 23 Class I and II).
- **Mounting:** Bottom of fuselage. Leveling is critical; physical "Zero Feet Offset" must be measured and programmed. Minimum 16-inch ground clearance.
- **Interfaces:** Connects via 22-pin connector, CAN bus, and serial. Fully integrates with Garmin GI 275 multifunction instruments (currently the only primary display interface for certified aircraft). Available for G3X Touch in experimental.

- Investment is described as "palatable" for owners already upgrading panels (Estimated market price ~$2,000).
