# Garmin PlaneSync: New School Avionics Housekeeping

**Source:** AVweb / Aviation Consumer
**Author:** Larry Anglisano
**Date:** April 2023
**Category:** Connectivity

Garmin PlaneSync is a new wireless system designed for remote communication with an aircraft's onboard avionics. It operates using the new GDL 60 LTE/Wi-Fi/Bluetooth unit in conjunction with the Garmin Pilot app. This system allows users to remotely perform various tasks, including loading navigation databases, checking fuel and aircraft voltage, offloading engine data, and viewing the aircraft's location on the airfield.

PlaneSync aims to enhance panel connectivity by enabling users to interact with their avionics from a distance using the Garmin Pilot app or a Garmin aviator watch. The GDL 60 datalink transceiver, which costs $3995, is central to the system, connecting the aircraft to a virtual cloud via Wi-Fi or cellular connection.

A PlaneSync subscription is also required, with an LTE North America plan priced at $30 per month (or $45 worldwide) for features like automatic database updates via LTE and Wi-Fi, auto flight log uploads, and remote aircraft status. A Wi-Fi-only subscription, which does not include LTE database updates or remote aircraft status, is available for $10 per month.
